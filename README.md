**Tech:** React · Node.js · Express.js · MongoDB · Mongoose · JWT · REST APIs  
**Features:** CV PDF import · Resume upload · Themes · Dark mode · View analytics · Portfolio publishing

#### 📚 Build Online Study Rooms
A full stack study platform designed for students to create online study rooms, manage tasks and collaborate in real time.

**Tech:** React · TypeScript · Node.js · Express.js · MongoDB · Socket.io  
**Features:** Authentication · Protected dashboards · Pomodoro timer · Notifications · Real-time collaboration

#### 🛒 ExpiryGuard
A Python desktop app for small retail businesses to manage product expiry dates, stock batches, discounts and potential waste.

**Tech:** Python · Tkinter · SQLite · SQL

**Features:** Booking flow · Route management · Fare records · Admin dashboard
